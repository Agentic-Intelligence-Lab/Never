# Continual Policy Distillation Project Page

This folder contains the standalone project page for:

**Continual Policy Distillation from Distributed Reinforcement Learning Teachers**

## Files

- `index.html`: the paper homepage, based on the repository academic project page template structure.
- `assets/paper.pdf`: local paper PDF used by the Paper button.
- `assets/framework.png`: web-friendly raster export of the main framework figure.
- `assets/framework.pdf`: original framework figure PDF used for the page and download button.
- `static/`: an unchanged copy of the template static assets so the page uses the original template CSS and JavaScript locally.

## How to Preview

Open `index.html` directly in a browser.

If you want to serve it locally instead:

```powershell
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000/continual_policy_distillation/
```
